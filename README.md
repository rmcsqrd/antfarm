## AntFarm
Force-based motion planning algorithm with RL navigational feedback.

### Usage
```
    $ cd path/to/your/directory
    $ julia
    $ julia> ]
    $ (v1.x) pkg> activate . 
    $ julia> import antfarm
    $ julia> antfarm.[command]()  // this is how it works
```

### Adding Modules to Project
```
    $ cd path/to/your/directory
    $ julia
    $ julia> ]
    $ julia> add [package]
```
